## ğŸ”¹ How Flask works (Simple Example)

`app.py`

```python
from flask import Flask

# Create a Flask app
app = Flask(__name__)

# Define a route
@app.route("/")

    return "Hello, Flask!"

# Another route with parameter

    return f"Hello, {name}!"

    app.run(debug=True)
```
